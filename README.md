# 🛒 BestBuy Data Management System - Product and Departments Manager

## Overview

**BestBuy Data Management System** is a console application developed using C# and Dapper ORM. This system efficiently manages product and department data for BestBuy, allowing users to retrieve, display, and manage various attributes of products and departments.

## Features

- 📊 **Data Management:** Retrieve and display all products and departments.
- ➕ **Product Creation:** Add new products with attributes such as name, price, category, stock level, and sale status.
- 🗂️ **Flexible Interfaces:** Designed with user-friendly interfaces for managing data.
- 🛠️ **Efficient Operations:** Utilizes MySQL for storage and Dapper ORM for streamlined database operations.

## Technologies Used

- **C#** - Programming language used for application development.
- **Dapper ORM** - Framework for efficient data access and manipulation.
- **MySQL** - Database system for storing product and department information.

## How It Works

1. **Connect to Database:** Establish a connection to the MySQL database.
2. **Manage Data:**
   - **Retrieve Products:** List all products with their details.
   - **Retrieve Departments:** List all departments with their details.
   - **Add Products:** Create new products by specifying attributes such as name, price, category, stock level, and sale status.
3. **Interface Design:** Use console-based interfaces to interact with the application and manage data effectively.
